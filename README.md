# Too Many Bats 

A is a game I made by following a tutorial by GDQuest. It is a 3D, single player game where you shoot bats and try to survive the time limit. I ran into a lot of complications while making this game, there are things from the tutorial that I couldn't include (like the puff clouds when a bat spawns and dies, and the text on the spawner). I plan to continue working on this game and make it much better than it is now

# Controls
- WASD or Arrow Keys to move
- X or left click to shoot
